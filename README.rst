Papis-dmenu
===========
|Pypi| |RTD| |Python| |TRAVIS|


This is a script that implements a GUI using the great `dmenu <https://tools.suckless.org/dmenu/>`_

Usage
-----

Just hit

::

  papis dmenu

Installation
------------

There is a version in PyPi

::

  pip3 install papis-dmenu

or

::

  sudo pip3 install papis-dmenu
